Value Analysis

가치분석 모형

Coombs & Meux 계열 · 국내: 가치원리 검사 + 사실 타당성

목표 · 초점

  • 개인의 입장 정당화: 사실 증거와 보편화 가능한 가치원리로 논증
  • 합리적·일관된 판단에 도달하기

전형적 절차

  • 문제 제시 → 가치문제 확인 → 사실 검증
  • 가치원리 검사: 새 사례 / 포함관계 / 역할교환 / 보편적 결과
  • 입장 수정·의사결정 → 적용

잘 맞는 주제

  • 생활·학급 맥락의 선택과 판단(정직↔배려, 휴대폰 규칙 등)

주의점

  • 자료의 중립성 전제와 검사 질문 설계 난도 ↑ → 교사 발문 훈련 필요
Jurisprudential Model

가치 갈등 해결(법리) 모형

Oliver & Shaver · 사회과 시민교육에서 발전

목표 · 초점

  • 공적 쟁점에서 정의·규범·사실·가치를 구분
  • 절차적 공정성을 통해 합리적 합의 도모

전형적 절차

  • 사태 제시 → 관련 규범·개념 명료화
  • 쟁점 유형화: 사실/용어/가치
  • 입장 제시·반론·수정 → 잠정 합의(절차 공정성)

잘 맞는 주제

  • 표현의 자유, 환경·안전, 지역 공익 등 공공정책 이슈

주의점

  • 원형이 사회과 모형 → 도덕 성찰이 옅어질 위험 → 원리 연결 분명히
Rational Decision

합리적 의사결정 모형

Engle–Frankel 결정교육 전통 수용 · 국내 개선안

목표 · 초점

  • 여러 대안 비교와 결과 예측, 기준 정립으로 최선 선택
  • 도덕적 판단력·문제해결력 신장

전형적 절차

  • 문제 제시·분석 → 관련 규범 의미·타당성 검토(연역/귀납)
  • 대안·결과 비교 → 선택·정당화 → 필요시 수정·잠정 결정

잘 맞는 주제

  • 생활 규범 결정, 학급 규칙 개정, 온라인 행동 선택 등

주의점

  • 결과주의로만 흐르지 않도록 규범·원리 검토 단계 명확히
Moral Discussion (Kohlberg)

도덕적 토론 수업 모형

도덕발달론(+1 전략) · 딜레마 토론

목표 · 초점

  • 판단력 수준 상승: 상위 단계 접촉·역할채택·논증

전형적 절차

  • 딜레마 제시 → 도입(문제 부각·왜 질문·상황 복잡화)
  • 심화(심층질문·역할채택·인접단계 논의) → 실천 동기 강화·확대 적용

잘 맞는 주제

  • 정의·책임·배려의 가치 충돌이 있는 도덕 딜레마

주의점

  • 학습자 발달수준·논증기술에 따라 인지적 과부하 가능 → 발문 난이도 조절

Value Analysis
가치분석		 Coombs & Meux 계열, 국내 도덕과에서 가치 원리 검사(4)와 사실 타당성 탐색을 핵심으로 정리·활용 개인이 자신의 입장을 사실 증거와 보편화 가능한 가치원리로 정당화하여 합리적 판단에 도달 문제 제시 → 가치문제 확인 → 사실 검증 → 가치원리 검사(새 사례/포함관계/역할교환/보편적 결과) → 입장 수정·의사결정 → 적용 생활·학급 맥락의 선택과 판단(정직↔배려, 휴대폰 규칙 등) ‘자료의 중립성’ 전제와 검사 질문 설계 난도 높음(교사 발문 훈련 필요)
Jurisprudential
가치 갈등 해결(법리)		 Oliver & Shaver의 법리적 탐구를 학교 수업에 맞게 재구성(사회과/시민교육에서 발전) 공적 쟁점에서 정의·규범·사실·가치를 구분해 절차의 공정성으로 합리적 합의를 도모 사태 제시 → 관련 규범·개념 명료화 → 사실/용어/가치 쟁점 구분 → 입장 제시·반론·수정 → 잠정 합의 표현의 자유, 환경·안전, 지역 공익 등 공적 정책 이슈 원형이 사회과 모형이라 절차만 남고 도덕 성찰이 옅어질 위험 → 도덕적 원리 연결 분명히
Rational Decision
합리적 의사결정		 Engle–Frankel 계열 결정교육 전통을 도덕과에 수용·개선(국내 개선안 제시) 대안 비교–결과 예측–기준 정립을 통해 최선의 선택 도출(도덕적 판단·문제해결력) 문제 제시·분석 → 관련 규범의 의미·타당성 검토(연역/귀납) → 여러 대안과 결과 비교 → 선택·정당화 → 필요시 수정·잠정 결정 생활 규범 결정, 학급 규칙 개정, 온라인 행동 선택 등 선택지 비교형 과제 결과주의로만 흐르지 않도록 규범·원리 검토 단계를 분명히 설계
Moral Discussion
도덕적 토론(콜버그)		 Kohlberg 도덕발달론(인지적 갈등·+1 전략) 기반의 딜레마 토론 도덕판단력 수준 상승(상위 단계 접촉·역할채택·논증)을 목표 딜레마 제시 → 도입(문제 부각·왜 질문·상황 복잡화) → 심화(심층질문·역할채택·인접단계 논의) → 실천 동기 강화·확대 적용 정의·책임·배려의 충돌을 담은 도덕 딜레마 학습자 발달수준·논증기술에 따라 인지적 과부하 가능 → 발문 난이도 조절

빠르게 구분하는 요령

  • 개인 선택의 정당화 훈련이면 → 가치분석 (증거 + 원리 검사)
  • 공적 쟁점의 공정 절차·합의면 → 법리모형 (쟁점 유형화·합의 절차)
  • 여러 대안 비교로 최선 결정이면 → 합리적 의사결정 (대안–결과–기준 체계화)
  • 딜레마로 판단수준 올리기면 → 도덕적 토론 (인지적 갈등·역할채택)
